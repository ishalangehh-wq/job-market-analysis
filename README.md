# 📊 Job Market Analytics Dashboard

## Project Overview
This project analyzes **12,000+ job records** to uncover:
- Most in-demand technical skills across industries
- Salary trends by job role
- Top hiring locations across India
- Job role distribution and market demand

**Tools Used:** Python · Pandas · SQL · Matplotlib · Tableau Public

---

## 🔗 Live Dashboard
👉 [View Tableau Public Dashboard](tableau_dashboard.png)

---

## 🔍 Key Insights
- **Python and SQL** were the most in-demand skills across all roles
- **Bangalore and Delhi** led all cities in hiring volume
- **ML Engineer** and **Data Scientist** roles commanded the highest average salaries
- Data Analyst demand was strong across both metro and Tier-2 cities

## 🛠️ Python Analysis
- Data loading and exploration using **Pandas**
- Skills frequency analysis using `split()` and `value_counts()`
- Salary benchmarking using `groupby()` and `mean()`
- Location-wise hiring analysis
- Visualizations using **Matplotlib** bar charts and pie charts

---

## 🗄️ SQL Analysis
Queries written using:
- `GROUP BY` · `COUNT()` · `AVG()` · `ORDER BY`

Analysis covered:
- Top job roles by frequency
- Average salary by job title
- Hiring volume by location
- Skill demand ranking
